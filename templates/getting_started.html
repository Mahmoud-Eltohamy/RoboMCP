Getting Started with MCP Appium

Learn how to set up and use MCP Appium for mobile automation testing

Installation

MCP Appium can be installed in several ways:

1. Using pip
pip install mcp-appium
2. Using Docker

We provide a Docker image with all dependencies pre-installed:

docker pull mcp-appium/mcp-appium:latest

Or use Docker Compose with the provided docker-compose.yml:

docker-compose up
3. From source
git clone https://github.com/mcp-appium/mcp-appium.git
cd mcp-appium
pip install -e .
Quick Start Guide
1. Start the Appium Server

First, make sure you have Appium installed and running:

appium
2. Install an Android Emulator or Connect a Device

You'll need an Android emulator or physical device connected to your system.

adb devices
3. Start the MCP Appium Server

Start the MCP Appium server with:

python main.py --server
4. Run a Simple Test

Try running one of the example scripts:

python examples/mock_example.py

Or for a real device test:

python examples/android_example.py
Using the AI Integration

MCP Appium comes with built-in AI capabilities powered by multiple providers:

Supported AI Providers:
  • OpenAI (GPT-4o) - For natural language interactions and test generation
  • Google Gemini - Alternative AI provider with different strengths
  • Hugging Face - For using open-source models via API
  • Ollama - For running local LLMs without external API dependencies
Setting Up AI Integration:

To use the AI integration features, set the appropriate environment variables:

# For OpenAI
export OPENAI_API_KEY=your_api_key

# For Google's Gemini
export GOOGLE_API_KEY=your_api_key

# For Hugging Face
export HUGGINGFACE_API_KEY=your_api_key

# For Ollama (local)
export OLLAMA_BASE_URL=http://localhost:11434
Sample Usage:

Here's a simple example of using the AI integration to generate a test script:

python examples/code_generation_example.py --provider openai --test-goal "Test the login functionality"
Multi-Language Support

MCP Appium can generate test scripts in multiple languages and frameworks:

Python

Using Appium Python Client

python examples/code_generation_example.py --language python
Java

Using Java-Client

python examples/code_generation_example.py --language java
JavaScript

Using WebdriverIO

python examples/code_generation_example.py --language javascript
C#

Using Appium.WebDriver

python examples/code_generation_example.py --language csharp
Ruby

Using appium_lib

python examples/code_generation_example.py --language ruby
Robot Framework

Using AppiumLibrary

python examples/code_generation_example.py --language robot
